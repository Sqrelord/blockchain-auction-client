## 基于区块链的拍卖交易系统

#### 构建

```shell
yarn install
```

#### 开发

```shell
# H5
yarn run dev:h5
# 微信小程序
yarn run dev:mp-weixin
```

#### 打包

```shell
# H5
yarn run build:h5
# 微信小程序
yarn run build:mp-weixin
```

#### 爬取数据

```shell
jest --runTestsByPath .\scraper\mock.scraper.test.js
```
